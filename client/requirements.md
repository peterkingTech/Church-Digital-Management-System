## Packages
@supabase/supabase-js | Core database and auth client
i18next | Internationalization framework
react-i18next | React bindings for i18next
recharts | Data visualization for dashboard
framer-motion | Animation library for smooth transitions
date-fns | Date formatting and manipulation
qrcode.react | QR code generation for invites
@fullcalendar/react | Calendar component
@fullcalendar/daygrid | Month view for calendar
@fullcalendar/interaction | Interaction plugin for calendar
clsx | Utility for constructing className strings
tailwind-merge | Utility for merging Tailwind classes

## Notes
- Using Supabase Client for all data fetching (no /api routes)
- Auth is handled via Supabase Auth
- Theme color is Amber (#f59e0b)
- Internationalization support for EN, ES, PT, FR, DE (defaults to EN for MVP)
